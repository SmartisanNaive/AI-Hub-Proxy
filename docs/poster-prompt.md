# AI Hub Proxy 二次元宣传图生成 Prompt

## 主 Prompt（中文输入图像模型）

```
二次元平面艺术插画，为科技项目 "AI Hub Proxy" 绘制的宣传图。

画面中央是一位原创的未来感少女/少年吉祥物，作为 AI 数据向导的形象。她身着墨蓝色（#1B365D）连帽外套，内衬暖米色（#f5f4ed），外套上带有细微的电路板纹理。人物不使用常规正面全身像，而是做出展现动态感的速写动作：单脚轻点地面，身体微微前倾，一只手向前伸展似乎在牵引或分发数据流，另一只手托举着一颗发光的半透明蓝色数据球，球体内部有星光流转。

画面采用 "窗口重叠 (Window Overlay)" 与 "数字拼贴" 构图。角色的身体轮廓由多个错位的矩形框构成，某些方框区域被处理成透明视窗，展示出清朗的蓝天与蓬松积雨云纹理，仿佛角色的体内蕴含着广阔的天空与无限的数据空间。从她的掌心延伸出几条发光的淡金色数据线，分别连接向画面四周悬浮的多个方形"门户窗口"，象征 AI Hub Proxy 将多个上游服务统一转发和中转。

画面中装饰有精美的故障艺术 (Glitch Art) 元素：极简的黑色几何长条、细密的彩色电子扫描线、错位的色彩偏移纹理（RGB split），以及漂浮的半透明小图标——对话框、锁、地球、API 括号 `{ }`、云朵节点——暗示 AI、安全、跨境访问、API 转发与云端代理。

整体视觉呈现现代平面设计的律动感，色彩以克莱因蓝（#002FA7）、纯净白和淡金色为主，背景简洁明快，氛围宁静且富有诗意。无文字，无水印，高分辨率，2D 平面动漫插画风格，适合作为项目落地页 Hero 图或社交媒体宣传图使用。
```

## 英文版（Midjourney / DALL·E / Stable Diffusion）

```
Anime flat art illustration, promotional artwork for the tech project "AI Hub Proxy".

Center stage is an original futuristic mascot girl/boy, appearing as an AI data guide. She wears an ink-blue (#1B365D) hooded jacket with warm cream (#f5f4ed) lining and subtle circuit-board patterns. Instead of a standard frontal full-body pose, she is captured in a dynamic sketch-like action: one foot lightly touching the ground, body leaning slightly forward, one hand reaching out as if pulling or distributing data streams, the other hand holding a glowing translucent blue data sphere with starlight flowing inside.

The composition uses "Window Overlay" and digital collage. The character's silhouette is formed by multiple misaligned rectangular frames; some frame regions are treated as transparent windows revealing a clear blue sky and fluffy cumulonimbus clouds, as if an expansive sky and infinite data space reside within her body. From her palm extend several glowing pale-gold data cables connecting to multiple floating square "portal windows" around the canvas, symbolizing how AI Hub Proxy unifies and forwards requests to multiple upstream services.

The image is decorated with refined Glitch Art elements: minimal black geometric strips, fine colorful scan lines, misaligned RGB split textures, and floating semi-transparent icons — chat bubble, lock, globe, API brackets `{ }`, cloud nodes — hinting at AI, security, cross-border access, API forwarding, and cloud proxy.

Overall visual rhythm of modern graphic design. Color palette dominated by Klein blue (#002FA7), pure white, and soft gold. Clean and bright background, serene yet poetic atmosphere. No text, no watermark, high resolution, flat 2D anime illustration style. Suitable as a Hero image for a project landing page or social media promotional artwork.
```

## 可选变体方向

在主 Prompt 后追加以下内容，可获得不同气质的宣传图：

- **更活泼动感**：`jumping pose, hair flowing in the wind, energetic smile, motion lines`
- **更神秘宁静**：`closed eyes, meditating floating pose, soft halo, dreamy clouds`
- **更科技冷峻**：`holographic blue glow, cyberpunk scan lines, serious expression, darker Klein blue shadows`
- **强调中转/跨境**：`data lines bridging two distant landmasses, passport-stamp motifs, airplane-path curves`
- **更扁平符号化**：`less detailed character, focus on geometric portals and data streams, Bauhaus-inspired`

## 推荐输出规格

- 比例：`16:9`（落地页 Hero 图）或 `4:5`（社交媒体）
- 背景：纯净白或极浅暖米色
- 最小尺寸：1920×1080 px

## 使用建议

生成后用于 kami 风格落地页时，建议把图片放在 Hero 区下方或作为大面积背景图使用。由于主色为克莱因蓝，与页面现有的墨蓝 `#1B365D` 属于同色系，可直接融入；如需完全一致，可在后期把克莱因蓝 `#002FA7` 略微向墨蓝 `#1B365D` 压暗。
