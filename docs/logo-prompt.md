# AI Hub Proxy 二次元 Logo 生成 Prompt

## 主 Prompt（中文输入图像模型）

```
一个二次元风格的科技公司 Logo，项目名为 "AI Hub Proxy"。

主体是一位坐在云端数据节点上的未来感少女/少年吉祥物，穿着带有电路板纹理的连帽外套，外套主色为墨蓝（#1B365D），内衬为暖米色（#f5f4ed）。角色单手托着一颗半透明、内部流转着星光的蓝色数据球，象征 "AI"；另一只手牵着几条发光的数据线，分别连接向身后的多个悬浮门户/传送门，象征 "Proxy" 与 "统一中转"。

背景是极简的浅色暖羊皮纸质感（#f5f4ed），点缀细微的淡金色网格线，营造科技地图感。角色周围漂浮着几个半透明小图标：对话框、锁、地球、API 括号，暗示 AI、安全、跨境、API 转发。

整体构图居中对称，留白充足，适合作为圆形或圆角矩形 App Icon 使用。线条干净，色彩克制，仅用墨蓝、暖米、淡金三种主色，无复杂渐变，无文字，无水印，纯白或透明背景，矢量插画质感，高分辨率，2D 平面动漫风格。
```

## 英文版（Midjourney / DALL·E / Stable Diffusion）

```
Anime-style tech logo for "AI Hub Proxy". A futuristic chibi mascot sitting on a cloud-like data node, wearing a hooded jacket with subtle circuit-board patterns in ink blue (#1B365D) and warm cream (#f5f4ed). The character holds a translucent blue data sphere glowing with stars in one hand, symbolizing AI, while glowing data cables extend from the other hand into floating portal gateways behind, symbolizing proxy and unified routing.

Background is minimal warm parchment (#f5f4ed) with faint gold grid lines, evoking a tech map. Small floating icons around the character: chat bubble, lock, globe, API brackets, hinting at AI, security, cross-border access, and API forwarding.

Centered, symmetrical composition with generous negative space. Suitable as a circular or rounded-square app icon. Clean linework, restrained palette of ink blue, warm cream, and soft gold. No text, no watermark, white or transparent background, vector illustration feel, high resolution, flat 2D anime style.
```

## 可选变体方向

若想要不同气质，可在主 Prompt 后追加：

- **更可爱**：`chibi proportions, big eyes, soft blush, kawaii mascot`
- **更科技冷峻**：`sharp angles, cyberpunk glow, holographic blue accents, serious expression`
- **更简约符号化**：`no character, abstract geometric logo: a central sphere with orbiting portals, line art style`
- **强调中转/跨境**：`data lines forming a bridge between two landmasses, passport-stamp motifs`

## 推荐输出规格

- 比例：`1:1`（App Icon）或 `4:3`（落地页 Logo）
- 背景：透明 PNG 或纯白
- 最小尺寸：1024×1024 px

## 使用建议

生成后如果需要放到 kami 风格的落地页上，建议把 Logo 做成圆角矩形（radius 8–16px），主色保持墨蓝 `#1B365D`，暖米色作为反白或镂空区域，与页面现有设计系统保持一致。
